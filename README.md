# Nostra - E-commerce Website

Nostra is a responsive e-commerce website built using **HTML, CSS, and JavaScript**. It includes a modern home page, collections page, and contact page with interactive features.

## Features
- Responsive design for desktop and mobile devices
- Product collections display
- Search functionality to find products quickly
- Filter functionality using JavaScript
- Contact form page
- Clean and user-friendly UI

## Technologies Used
- HTML5
- CSS3
- JavaScript
